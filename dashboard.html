Anlisis de Ventas De Una Empresa Tecnolgica

Este proyecto realiza un anlisis integral de los datos de ventas y envos de una empresa dedicada a la comercializacin y distribucin de productos Tecnolgicos. A partir de archivos CSV, se emplean herramientas de Python, como Pandas y Matplotlib, para procesar, visualizar y comprender la informacin clave del negocio.

Los principales indicadores analizados incluyen:

  • Rendimiento general de ventas
  • Rubros ms rentables
  • Clientes con mayor volumen de compra
  • Evolucin mensual de las ventas
  • Productos ms vendidos
  • Promedio de reseas y desempeo de envos

El objetivo de este anlisis es ofrecer una visin clara, precisa y visualmente informativa del rendimiento comercial, facilitando la toma de decisiones estratgicas basadas en datos.

Carga de datos

Primero se importan las libreras necesarias y se cargan los archivos CSV desde la carpeta csv de ventas, que contiene toda la informacin del sistema (clientes, facturas, productos, rubros, detalles de facturas y resea).

Tienen que existir los cvs en la carpeta "csv de ventas".

In[11]:
import os
import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

base_path = "csv de ventas_2025/"

archivos = ["cliente.csv", "factura.csv", "detalle_factura.csv", "producto.csv", "rubro.csv", "resea.csv"]
faltantes = [f for f in archivos if not os.path.exists(os.path.join(base_path, f))]

if faltantes:
    mensaje = " ERROR: No se encontraron los siguientes archivos requeridos:\n"
    for f in faltantes:
        mensaje += f"   - {f}\n"
    mensaje += "\n Revis la carpeta 'csv de ventas_2025' antes de ejecutar el anlisis."
    raise FileNotFoundError(mensaje)
else:
    # Cargar los CSV una sola vez si todos existen
    clientes = pd.read_csv(base_path + "cliente.csv")
    facturas = pd.read_csv(base_path + "factura.csv")
    detalle = pd.read_csv(base_path + "detalle_factura.csv")
    productos = pd.read_csv(base_path + "producto.csv")
    rubros = pd.read_csv(base_path + "rubro.csv")
    reseas = pd.read_csv(base_path + "resea.csv")

Preparacin de los datos

Se combinan las tablas mediante sus claves para crear un nico DataFrame que contenga toda la informacin consolidada. Adems, se calcula el subtotal de cada producto y se formatea la fecha correctamente.

In[12]:
df = detalle.merge(productos, on="id_producto", how="left")
df = df.merge(rubros, on="id_rubro", how="left")
df = df.merge(facturas, on="id_factura", how="left")
df = df.merge(clientes, on="id_cliente", how="left")

df = df.merge(reseas[['id_factura', 'puntaje', 'fecha_resea']], on="id_factura", how="left")

df["subtotal"] = df["cantidad"] * df["precio_unitario"]
df["fecha"] = pd.to_datetime(df["fecha"], errors="coerce")
df["fecha_resea"] = pd.to_datetime(df["fecha_resea"], errors="coerce")

reseas["fecha_resea"] = pd.to_datetime(reseas["fecha_resea"], errors="coerce")

reseas_2025 = df[df["fecha_resea"].dt.year == 2025].copy()
if not reseas_2025.empty:
    reseas_unicas_2025 = reseas[reseas["fecha_resea"].dt.year == 2025]
    
    reseas_mensuales = reseas_unicas_2025.groupby(reseas_unicas_2025["fecha_resea"].dt.month).agg({
        'puntaje': 'mean',
        'id_factura': 'count'
    }).reindex(range(1,13), fill_value=0)
    
    meses_con_reseas = reseas_mensuales[reseas_mensuales['id_factura'] > 0]
    
    fig, ax1 = plt.subplots(figsize=(10, 5))
    
    color_linea = 'red'
    ax1.plot(range(1,13), reseas_mensuales['puntaje'], 
             marker='o', color=color_linea, linewidth=3, markersize=6, label='Puntaje Promedio')
    ax1.set_ylabel('Puntaje Promedio (1-5)', color=color_linea)
    ax1.set_ylim(0, 5.5)
    ax1.tick_params(axis='y', labelcolor=color_linea)
    ax1.grid(True, alpha=0.3)
    ax1.set_xticks(range(1,13))
    ax1.set_xticklabels(meses)
    
    ax2 = ax1.twinx()
    color_barras = 'skyblue'
    barras = ax2.bar(range(1,13), reseas_mensuales['id_factura'], 
                    alpha=0.6, color=color_barras, label='Cantidad de Reseas')
    ax2.set_ylabel('Cantidad de Reseas', color=color_barras)
    ax2.tick_params(axis='y', labelcolor=color_barras)
    
    for i, valor in enumerate(reseas_mensuales['id_factura']):
        if valor > 0:
            ax2.text(i+1, valor + 0.1, f'{valor:.0f}', 
                    ha='center', va='bottom', fontsize=8, color=color_barras)
    
    for i, valor in enumerate(reseas_mensuales['puntaje']):
        if valor > 0:
            ax1.text(i+1, valor + 0.1, f'{valor:.1f}', 
                    ha='center', va='bottom', fontsize=9, color=color_linea, fontweight='bold')
    
    plt.title("EVOLUCIN DE RESEAS DE ENVOS (2025)\nPuntaje Promedio y Cantidad Mensual")
    
    lines1, labels1 = ax1.get_legend_handles_labels()
    lines2, labels2 = ax2.get_legend_handles_labels()
    ax1.legend(lines1 + lines2, labels1 + labels2, loc='upper left')
    
    plt.tight_layout()
    plt.show()
    
    puntaje_promedio_total = reseas_unicas_2025['puntaje'].mean()
    total_reseas = len(reseas_unicas_2025)
    
    if not meses_con_reseas.empty:
        mejor_mes = meses_con_reseas['puntaje'].idxmax()
        peor_mes = meses_con_reseas['puntaje'].idxmin()
        
        print(f"\n MTRICAS DE RESEAS DE ENVOS:")
        print(f"    Puntaje Promedio General: {puntaje_promedio_total:.2f}/5")
        print(f"    Total de Reseas Recibidas: {total_reseas}")
        print(f"    Mejor Mes: {meses[mejor_mes-1]} ({meses_con_reseas.loc[mejor_mes, 'puntaje']:.2f}/5)")
        print(f"    Mes a Mejorar: {meses[peor_mes-1]} ({meses_con_reseas.loc[peor_mes, 'puntaje']:.2f}/5)")
    else:
        print(f"\n MTRICAS DE RESEAS DE ENVOS:")
        print(f"    Puntaje Promedio General: {puntaje_promedio_total:.2f}/5")
        print(f"    Total de Reseas Recibidas: {total_reseas}")
        print(f"    No hay suficientes datos para determinar mejor/peor mes")
